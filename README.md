# Azul in Python

## Project Description
This project is a Python implementation of the board game **Azul**.  
The goal is to recreate the core game mechanics in code, including tile selection, pattern lines, wall tiling, scoring, floor penalties, and endgame bonuses.

## Features
- Supports the main rules of Azul
- Tile drafting from factories and the center of the table
- Pattern line placement validation
- Wall tiling and score calculation
- Floor line penalties
- End-of-round and end-of-game logic

## Technologies Used
- Python 3

## Objective
The purpose of this project is to practice object-oriented programming, game logic design, and rule implementation by building a playable digital version of Azul.
